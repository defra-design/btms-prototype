Search

  • You can search by using just the last 7 digits of a CHED reference as well as by the other methods listed on the search page.
  • Search results will only cover the last 4 months.

Results page

  • When a reference is in red and has a dash underline it means if you hover your cursor over it, you’ll find more information about the reference.
  • On customs declaration results you can use the filters at the top to make the search results more specific. You can filter by matches, authority and decision.
  • On IPAFFS notification (CHED) results you can filter by authority.

Latest activity

  • This page shows the date and time that the last customs declaration and finalisation messages were received from CDS.
  • This is intended to help you identify that BTMS is working as it should and there are no problems.

Account management

If a colleague needs access to BTMS:

  • For colleagues with Defra, APHA or FSA email addresses, they should request BTMS access through the My IT portal.
  • For colleagues with HMRC or PHA email addresses, they should ask their local BTMS administrator to grant access.
